# Notebook LLM Exports

Research notes, analysis, and summaries generated by [Google Notebook LLM](https://notebooklm.google/).

## Guidelines

- **Markdown** (`.md`) → commit to this directory
- **PDFs** (`.pdf`) → upload to [GitHub Releases](https://github.com/Z0mb13V1/mindcraft/releases), then link from the Markdown summary
- PDFs in this directory are **gitignored** to avoid bloating the repo

## Naming Convention

```
YYYY-MM-DD_topic-slug.md       ← Markdown summary (tracked)
YYYY-MM-DD_topic-slug.pdf      ← PDF export (gitignored → upload to Releases)
```

Example:
```
2025-01-15_ensemble-pipeline-analysis.md
2025-01-15_ensemble-pipeline-analysis.pdf   ← gitignored
```
